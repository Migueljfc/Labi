Dhrystone

O Dhrystone é um software de benchmark desenvolvido por Reinhold Weicker é um programa de teste sintético que avalia o desempenho do processa- dor não tendo conta em operações de vírgula flutuante.


Como usar?

Para usar o Dhrystone no Ubunto basta guardar o "source code" disponivel aqui e depois correr no terminal os seguintes comandos.Deveremo executar estes comandos dentro dos diretórios onde o ficheiro guardado se encontra.

Ubuntu

gcc "nome do ficheiro".c ./a.out

Windows

gcc "nome do ficheiro".c "nome do ficheiro criado".exe
