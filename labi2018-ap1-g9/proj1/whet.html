Whetstone

O Whetstone é um software de bechmark para avaliar o desempenho dos compu- tadores, ao contrário do Dhrystone, o Whetstone é capaz de avaliar a capacidade do CPU de fazer operações com números "floating point".


Como usar?

Para usar o Whetstone no Ubunto basta guardar o "source code" disponivel aqui e depois correr no terminal os seguintes comandos. Deveremo executar estes comandos dentro dos diretórios onde o ficheiro guardado se encontra.

Ubuntu

gcc "nome do ficheiro".c -lm ./a.out

Windows

gcc "nome do ficheiro".c "nome do ficheiro criado".exe
